# 个人记账系统 C++ 控制台版
## 项目简介
基于C++控制台开发的本地记账工具，使用vector存储账单数据，搭配txt文件持久化存储，支持账单增删、收支统计、本地文件读写，界面自动清屏，完善异常捕获防闪退。

### 核心设计特点
1. **双层ID分离机制**
- 用户可见连续展示序号：删除账单后自动补齐，永远无断层，操作直观
- 底层永久唯一ID：仅存于bill.txt，每条账单终身独有，用于底层溯源，终端不展示
2. 数据持久化：程序关闭自动保存，启动自动读取历史账单
3. 全流程异常拦截：非法字符输入、文件读写失败、越界删除均捕获，不会程序闪退
4. Windows控制台自动清屏，操作后按回车刷新菜单，界面整洁

## 功能菜单
=====个人记账系统=====
1.新增账单
2.查看全部账单（展示序号永久连续）
3.根据展示序号删除账单
4.收支统计
5.保存账单到文件
0.退出程序

### 功能介绍
1. **新增账单**
输入账单名称、金额（正数=收入，负数=支出），自动分配连续展示序号，底层生成永久唯一ID存入容器。
2. **查看全部账单**
打印所有账单连续序号、名称、金额，底层ID仅保存在bill.txt，界面不显示。
3. **删除账单**
输入界面展示的连续序号，直接按下标删除，删除后后续账单序号自动前移补齐，无断层。
4. **收支统计**
自动计算总收入、总支出、结余金额。
5. **保存账单到文件**
将当前所有账单写入bill.txt持久化存储。
6. **退出程序**
可选自动保存账单后关闭程序。

## 编译运行教程
### 运行环境：Dev-C++ Windows
1. 双击作业.dev打开项目
2. 确认项目包含main.cpp、test.h两个核心文件
3. 点击顶部「运行-编译运行」，自动生成作业.exe
4. 运行后同目录自动生成bill.txt存储账单数据

### 系统适配说明
- 清屏函数system("cls")仅适配Windows系统；Linux/Mac需替换为system("clear")
- 文件编码默认ANSI，中文无乱码

## 数据文件 bill.txt 格式
文件每行存储3项数据，空格分隔：底层唯一ID 账单名称 金额
示例：
1 工资 6000
2 午餐 -25
3 打车 -12
> 注意：手动修改/删除txt文件会破坏数据，建议仅通过程序操作账单。

## 异常容错机制
1. 输入字母、符号等非数字菜单指令，自动拦截报错，不会闪退
2. 删除不存在的账单序号、空账单名称、文件读写失败均弹出友好提示
3. 全局顶层try-catch兜底捕获未知致命异常，程序安全退出
4. 读取文件时自动过滤非法脏数据，避免容器越界崩溃


## 注意事项
1. 删除bill.txt会清空所有账单记录，程序重启后重新创建空文件
2. 编译缓存main.o、布局文件作业.layout可直接删除，不影响程序逻辑
3. 底层真实ID只会递增，删除账单不会复用旧ID，保证每条账单数据唯一性
4. 设计解耦：仅修改accountbook账本类，bill账单类全程无改动，拓展性强
